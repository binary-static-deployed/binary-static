ログインにお困りの場合

ご口座へのログインが上手くいかない場合は、ブラウザの設定からCookieを有効に設定して下さい。当社サイトでは、ログイン時にお客さまのログインIDと暗号化されたパスワードをお客さまのコンピュータに保存する目的でCookieを利用しています。

ご利用中のブラウザのバージョンの指示に従ってクッキーを有効化して下さい。

Binary (C.R.) S.A.

Chrome 40+

  • ブラウザの"クロムメニュー" のドロップダウンメニューから設定を選択して下さい。
  • そして詳細設定を表示をクリックして下さい。
  • 次にプライバシー セクションのコンテンツ設定ボタンをクリックして下さい。
  • ローカルへのデータ設定を許可する(推奨)を選択して下さい。
  • 最後に終了ボタンをクリックし、設定を保存して下さい。
Binary (IOM) Ltd

Mozilla Firefox 40+

  • メニューを開いて Preferencesを選択して、
  • プライバシータブをクリックして下さい。
  • 設定画面にて"Firefox"に"履歴を記憶させる"と設定をして下さい。
  • その後、OKボタンをクリックし設定を保存して下さい。
Binary (Europe) Ltd

Safari 5.x+

  • まず初めにSafariメニューへいき、Preferencesを選択してください。
  • 次にプライバシータブをクリックしましょう。
  • そして、"クッキーをブロックする"項目をNeverへ設定しましょう。
  • 最後に右上のXボタンをクリックして、設定が保存されるのでページを閉じましょう。
Binary (C.R.) S.A.

Opera 30+

  • まず初めにToolsメニューにアクセスし、Preferencesを選択しましょう。
  • そしてAdvancedタブをクリックしましょう。
  • 次にCookiesを左側の欄から選択しましょう。
  • そしてAccept cookiesを選択しましょう。
  • 最後にOKボタンをクリックし、設定を保存してページを閉じましょう。
Binary Ltd

Microsoft Internet Explorer 11+

  • ブラウザのツールメニューからインターネットオプションを選択して下さい。
  • プライバシータブを選択して下さい。
  • 詳細設定ボタンをクリックして下さい。
  • 自動Cookie 処理を上書きするにチェックを入れて下さい。
  • ファーストパーティのCookieとサードパーティのCookieを両方ともに受け入れるとマークして下さい。
  • OKボタンをクリックし、設定を保存してウィンドウを閉じて下さい。
Binary Ltd

Microsoft Edge 12+

  • まず初めにその他アクションメニューにアクセスし、設定を選択しましょう。
  • そして詳細設定にある詳細設定を確認するボタンをクリックしてください。
  • 次にクッキーセクションから"クッキーをブロックしない"オプションを選択します。
  • 表示されているメニュー外をクリックすると保存されますので、終了してください。

■店頭バイナリーオプション取引は金融商品取引法に規定される店頭デリバティブ取引であり、元本や利益が保証されていません。■店頭バイナリーオプション取引は、判定時刻に自動的に権利行使されるヨーロピアンタイプと、取引期間中に決められた条件に達すると権利が自動行使されるアメリカンタイプがあります。■店頭バイナリーオプションは、外国為替相場等の変動により投資元本の全てを失うおそれのあるリスクの高い金融商品です。■判定時刻及び取引期間中における一定の権利を、オプション料を支払って購入する期限がある取引のため、権利が消滅した場合には支払ったオプション料の全額を失います。■購入価格は1ロット1,000円〜100,000円です。■購入価格と売却価格にはスプレッドがあり、相場情勢により拡大することがあります。■相場情勢により、お客様が希望した価格より不利な価格で取引が成立する場合があります。■店頭バイナリーオプションの取引上限は以下のとおりです。 1)1回の取引でペイアウトは100,000円まで。 2)1日の取引でペイアウトは10,000,000円まで 3)アメリカンタイプ（タッチ・ノータッチ、レンジ）の1日の制限は a) 予想される利益額は5,000,000円まで b) 確定された利益額は5,000,000円まで c) 取引数量は5,000,000円まで 　上記いずれかに達した時点で、お客さまはその日の取引ができなくなります。 ■取引手数料は無料ですが、12ヶ月間、新規又は決済の取引がなかった場合は口座維持管理料（年間2,500円）が発生します。■オプションの購入後の注文取り消しは行う事ができませんが、取引可能期間であれば売却は可能です。ただし、購入価格と売却価格にはスプレッドがあり、売却時に損失を被る可能性があります。■お取引を開始される前に、契約締結前交付書面を熟読の上、必ず約款及び取扱説明書に書かれてある取引の特徴、取引条件、仕組み及びリスクを十分確認いただき、ご自身の判断と責任において、自己の計算によりお取引ください。
