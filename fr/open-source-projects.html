Binary.com projets open source

Binary.com est partisan actif du mouvement open source et est dans le processus d'ouvrir au public un nombre des composants de sa base de codes.

Contenu statique

Améliorez le contenu frontal de Binary.com

Le contenu statique du site de Binary.com (CSS, Javascript) est disponible sous le binary-static projet open source sur github. Vous voulez remodeler le site Binary.com, ou améliorer la comptabilité du navigateur, la vitesse du rendu du site ou la performance Javascript? Veuillez fourcher le projet et soumettre des demandes pull de vos suggestions de changement de code.

Modules Perl

Contributions CPAN de Binary.com

Binary.com est un membre actif de la communauté Perl et est dans le processus de télécharger un nombre de ses modules internes d'utilités vers CPAN. Si vous êtes un enthousiaste de Perl, contribuez en révisant ces modules, et en soumettant des demandes de pull sur github.
