Começar a negociar com a MetaTrader 5

  1. Click the button below to download MetaTrader.
  2. Run the installation file and follow the instructions.
  3. Log into the application using your MetaTrader account number (e.g., 1124) and your MetaTrader account password.

If you previously downloaded MetaTrader 5 directly from MetaTrader’s website, please select “Binary.com” from the list of available brokers.
